# 📚 Library Application

## 🚀 Live Demo
**[👉 View Live Application](https://library-app.onrender.com)**

## 📋 Features
- ✅ Add, view, delete books
- ✅ PostgreSQL database
- ✅ Basic frontend
- ✅ REST API

## 🛠️ Tech Stack
- Java 21, Spring Boot 3, PostgreSQL
- HTML, JavaScript, CSS
- Maven, Docker

## 🏗️ Architecture
- **Backend**: Spring Boot REST API
- **Frontend**: Vanilla JavaScript
- **Database**: PostgreSQL
- **Deployment**: Render + Docker
